# Warm ups

Prepares body and mind for the main session.

RAMP warm ups:
- Raise, heart rate, temp, blood flow, muscle elasticity
- Activate, muscles for main session
- Mobilise, joints and tendons with similar movements to those in main session
- Potentiate, increase the stress and intensity. Prepares body and mind for main session

Intensity should increase progressively.

## Fundamentals

- Lunge
- Press up
- Squat
- Plank
- Jump off
- Landing jumps
